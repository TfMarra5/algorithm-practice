Outlines the logic behind an Amazon-style recommendation system using:

- **Collaborative filtering**
- **Content-based filtering**
- **Hybrid approaches** combining multiple strategies

### 6. Personalized Media Suggestions

Designs an algorithm for a Netflix-like movie suggestion system using machine learning concepts such as:

### 9. Conceptual Questions (Tech-Oriented Theory)

**A) Types of Algorithms**  
Description of deterministic, probabilistic, heuristic, parallel, and machine learning-based algorithms, including real-world applications in:

- Finance
- Artificial Intelligence
- Logistics
- Healthcare

**B) Recursive vs Iterative Algorithms**  
Comparison of recursion and iteration, including:

- Performance implications in large-scale systems
- Use cases such as tree traversal in databases and graph search in AI
- Memory considerations in constrained environments

**C) Dynamic Programming vs Greedy Algorithms**  
Explanations supported by industry examples:

- DP in supply chain and scheduling optimization
- Greedy strategies in routing, resource allocation, and approximation algorithms

**D) Divide and Conquer**  
Concept and real-world applications such as:

- QuickSort in database systems
- MapReduce in big data processing
- Parallel computation in cloud environments

**E) Data Structures & Algorithm Efficiency**  
Discussion on how choosing between arrays, trees, graphs, hash maps, tries, and heaps impacts:

- AI systems
- Database indexing
- Blockchain design
- Real-time and low-latency systems

**F) Algorithm Complexity**  
Overview of time and space complexity, extended to:

- API latency
- Throughput in streaming systems
- Memory usage on edge devices
- Cost and scalability in cloud computing
